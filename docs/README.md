## Documentation Best Practices

When updating documentation:

1. **Keep it up-to-date**: Update docs when code changes
2. **Be clear and concise**: Use simple language and short sentences
3. **Provide examples**: Include code samples and screenshots
4. **Organize logically**: Use headings and table of contents
5. **Link related docs**: Cross-reference related documentation
